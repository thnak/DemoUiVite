## Color Scheme

- **Shift Time** (Primary/Blue): Regular working hours
- **Break Time** (Warning/Orange): Lunch, tea breaks, rest periods
- **Overtime** (Info/Light Blue): Extra working hours beyond regular schedule
- **Planned Stop** (Error/Red): Planned maintenance, shutdowns, holidays

## View Types

### 1. Date Range View
- User selects custom start and end dates
- Displays all periods within the selected range
- Best for viewing specific time windows

### 2. Month View
- User selects year and month
- Automatically sets date range to the selected month
- Best for monthly planning and review

### 3. Year View
- User selects a year
- Automatically sets date range to the entire year
- Best for annual overview and planning
